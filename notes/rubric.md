# NexusDojo Skill Rubric

Pillars tracked: Python fundamentals, CLI/tooling, APIs, Testing. Scores: 1=Foundations, 3=Proficient, 5=Hiring-ready.

## Python fundamentals
- 1: Write functions, handle input/output, basic error handling.
- 3: Structure modules, handle files/args/env vars cleanly, add docstrings, handle edge cases.
- 5: Design small packages, type hints, logging, graceful failure and recovery paths.

## CLI/tooling
- 1: Use argparse, parse flags, print helpful output.
- 3: Handle subcommands, config paths, basic validation, friendly errors.
- 5: Extensible commands, config files, good UX/help, safe defaults/guards.

## APIs
- 1: Build a simple HTTP endpoint and parse JSON.
- 3: Validate inputs/outputs, handle errors, write basic tests.
- 5: Design small API surfaces, error models, middlewares, observability hooks.

## Testing
- 1: Run unit tests locally, assert behavior.
- 3: Structure tests per module, fixtures, cover happy + edge cases.
- 5: Good coverage, fast tests, clear failure messages, basic property-style checks.
